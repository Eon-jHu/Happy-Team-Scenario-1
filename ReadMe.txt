#HOSPITAL SIMULATOR:
This application is a rough implementation using debug messages of a hosptial simulator game in Unity. This is a first year project for Media Design School.

##Known bugs -
No known bugs as of 10/11/2023

#CODE STYLE:
Camel Case

#CONTROLS:
Purchase a Nurse: Press N
Purchase a Doctor: Press D
Fire a Nurse: Press B
Fire a Doctor: Press S
Bring Nurse Near/Away from Patients: Press P
